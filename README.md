# Pokédex 🔴

A Pokédex web app built with vanilla **HTML, CSS and JavaScript**, consuming the [PokéAPI](https://pokeapi.co/) to display the first 251 Pokémon with filtering by type.

---

## ✨ Features

- Displays the first 251 Pokémon fetched from the PokéAPI
- Filter by type: Normal, Fire, Water, Grass, Electric, Ice, Fighting, Poison, Ground, Flying, Psychic, Bug, Rock, Ghost, Dark, Dragon, Steel, Fairy
- Shows each Pokémon's official artwork, ID, name, type badges, height and weight
- Clean responsive design with type-based color coding

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling & type color themes |
| JavaScript (Vanilla) | Logic & DOM manipulation |
| PokéAPI | Pokémon data source |

---

## 🚀 Getting Started

No dependencies or build tools required — just open the file in your browser.

```bash
git clone https://github.com/yopukka/pokedex.git
cd pokedex
open index.html
```

Or simply drag `index.html` into your browser.

---

## 🔌 API

This project uses the free [PokéAPI](https://pokeapi.co/):

```
https://pokeapi.co/api/v2/pokemon/{id}
```

Data used per Pokémon: name, id, types, sprites (official artwork), height, weight.
